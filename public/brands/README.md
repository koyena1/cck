# Brand Logos Directory

Place brand logo images in this directory with the following names:

1. `hikvision.png` - Hikvision logo
2. `cpplus.png` - CP Plus logo
3. `dahua.png` - Dahua logo
4. `prama.png` - Prama logo
5. `secureye.png` - Secureye logo
6. `zebronics.png` - Zebronics logo
7. `daichi.png` - Daichi logo
8. `godrej.png` - Godrej logo

## Recommended Image Specifications:
- **Format**: PNG with transparent background
- **Size**: 200x200 pixels (or square ratio)
- **Quality**: High resolution for crisp display
- **Background**: Transparent or white

## Where to find logos:
- Download official logos from each brand's website
- Look for "Press Kit" or "Media Resources" sections
- Use brand guidelines to ensure proper logo usage

## Fallback:
If a logo is not available, the system will display a placeholder image.
